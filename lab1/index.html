Step 1: Setting up your workspace

Navigate to a folder in your desktop where you would like to store this handson. (As you can see in the example above, I just set it up in a folder called "workspace") Create a folder called handson1. We will store everything we need for this website in that folder.

Now, navigate into your handson1 folder, and create a folder called assets. Remember, this is where we will store our CSS and Javascript files, as well as images and any additional files.

Navigate into the assets folder, and create two new folders: one titled css, and one titled img. We won't be adding any CSS styles in this activity (that's for tomorrow!), but we will be inserting an image.

Either find an existing image on your computer, or Google Search for an image of your favorite animal. Copy it into the img folder you just created. We will be adding this to our HTML site later.

Open up your text editor, and create a new file. Before you do anything else, save this file in your handson1 folder as index.html (Note: this should be saved on the same level as your "assets" folder - not inside of it!).

Now, you should have a workspace that looks something like this:



Step 2: Structuring your file

Setting up your HTML file correctly is important - if you don't do it right, you won't have a working website!

You should still have your index.html file open in your text editor. Declare this page as an HTML file, then set it up as an HTML document.

Add a title to your site! In the head of your HTML file, add the title My First Website

Navigate to your workspace on your desktop (open the folder handson1) and open up your index.html file in Google Chrome. You shouldn't see any content on your website, but check that the tab in your window has the title you've just added.

Step 3: Adding a heading

HTML has tags that allow you to create headings of different sizes. On the first line inside the body of your file, add your favorite food, and make it a heading of the largest possible size.

Hint: The heading tags are h1, h2, h3, h4, h5, and h6.

Step 4: Adding a subheading

Now you want to add a subheading that is slightly smaller than your heading. On the next line, add your favorite song at the moment, and make it a heading of the next largest size.

Refresh your site on Google Chrome. You should now see something that looks like this:



Step 5: Adding a link

On the next line, write This site is dope, and make it so that This site links to a website you think is really cool.

Step 6: Adding a paragraph

HTML ignores white space: in other words, adding extra lines between text in your HTML file won't actually translate to line separations on your website. In order to create space between paragraphs, you will have to use a specific HTML tag.

Right under your website link, create a paragraph using the correct HTML tag. Write a sentence or two explaining why you like the website. To fill the rest of your paragraph, add some filler text (A lot of web designers will use what's called "Lorem ipsum" text as dummy text).

If you're using Sublime as your text editor, you can just type the word "lorem", then press the Tab button. It should generate a paragraph of lorem ipsum! (Otherwise, check out these fun lorem ipsum generators if you want to mix things up.)

Go back to Google Chrome, where your website should be open. Refresh the page - and your site should now look like this:



Step 7: Adding an image

Remember the image we saved in our img folder? Add this image under your paragraph. Because it's a local folder stored on your desktop, you want to use a relative path.
